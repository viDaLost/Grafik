## Как установить

### 1. Создайте Apps Script проект

Откройте `script.google.com` и создайте новый проект.

### 2. Добавьте файлы из архива

Вставьте содержимое:
- `Code.gs`
- `Index.html`
- `appsscript.json`

### 3. Укажите токен Telegram-бота

В Apps Script откройте:

`Project Settings` → `Script properties`

Добавьте свойство:

- `TELEGRAM_BOT_TOKEN` = `ВАШ_ТОКЕН_БОТА`

Это нужно для серверной проверки `Telegram.WebApp.initData`.

Telegram рекомендует доверять данным Mini App только после проверки `initData` на сервере, а Apps Script web apps публикуются через `doGet()`/HTML Service и могут общаться с серверными функциями через `google.script.run`. citeturn869640view0turn226916search4turn226916search0

### 4. Разверните как Web App

В Apps Script:

`Deploy` → `New deployment` → `Web app`

Параметры:
- **Execute as:** `Me`
- **Who has access:** `Anyone`

Apps Script web apps публикуются через `doGet()`/`doPost()` и возвращают HTML через HtmlService. citeturn226916search4turn498817view1turn226916search5

### 5. Подключите URL в Telegram-бота

Откройте BotFather и добавьте вашему боту кнопку mini app / web app, которая открывает URL вашего деплоя.

Важно: приложение должно открываться именно как Telegram Mini App, чтобы Telegram передал `initData`, а не просто как обычная ссылка. Telegram отдельно предупреждает, что `initDataUnsafe` нельзя считать доверенным и сервер должен валидировать `initData`. citeturn498817view0turn869640view1

## Что умеет приложение

- выбрать дату занятия
- отметить, кто был / не был
- указать причину отсутствия и комментарий
- сохранить отметки в Google Sheets
- просматривать историю занятий
- смотреть матрицу посещаемости по месяцам
- добавлять учеников
- удалять учеников из списка `Students` и полностью убирать их записи посещаемости из Google таблицы
- ограничивать доступ двумя Telegram ID
